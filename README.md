## 特色

- **完全白紙起點**：從「什麼是向量」教起，不假設任何前置知識
- **第一性原理**：每個概念都從「為什麼需要它」出發，推導到數學公式
- **矩陣級拆解**：所有維度變化都精確追蹤，不跳步
- **故事驅動**：用問題-解決的敘事結構串聯所有概念
